# mindiweik.com

Mindi Weik's personal site: blog, [WIP] podcast, speaking, and projects. Built with Astro.

## Develop
```bash
npm install
npm run dev
```

## Build
```bash
npm run build   # static output in dist/
```

Content lives in `src/content/`. See [CLAUDE.md](./CLAUDE.md) for the design system and
conventions.
